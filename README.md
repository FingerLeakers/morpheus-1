### How to

```
    options:
            --host,     -h      <hostname>
            --user      -u      <username>
            --wordlist  -w      <wordlist>
            --protocol  -p      <protocol>
    
    Protocols:
            ftp
            ssh
            
    Example:

        python morpheus.py --host 192.168.1.1 --user root --wordlist wordlist.txt --protocol ssh
```
