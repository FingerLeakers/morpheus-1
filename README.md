### fuckFTP
fuckFTP is a tool for brute force on ftp servers, allowing upload file on the server when the attack is finished.

### How to

```
    options:
            --host      <hostname>
            --user      <username>
            --wordlist  <wordlist>

    Example:

        python fuckFTP.py --host 192.168.1.1 --user root --wordlist wordlist.txt
```
